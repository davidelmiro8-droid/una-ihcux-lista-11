Proposta de valor

O principal benefício do ReCiclo é facilitar a doação e reutilização de itens usados, conectando pessoas que desejam descartar objetos com outras que precisam deles. O aplicativo promove sustentabilidade, reduz o desperdício e incentiva o consumo consciente de forma simples e acessível.

Descrição das telas

Tela de Home/Feed Exibe os itens disponíveis para doação, como roupas, móveis e bicicletas. O usuário pode navegar pelas opções ou aplicar filtros.

Tela de Filtros Permite refinar a busca por categoria (ex: roupas, móveis), estado do item (novo, semi-novo, usado) e outros critérios.

Tela de Detalhes do Item Mostra informações completas do produto, como descrição, estado e imagens. A partir daqui, o usuário pode demonstrar interesse e iniciar uma conversa.

Tela de Chat Espaço de comunicação entre doador e interessado. Permite tirar dúvidas e combinar retirada do item.

Tela de Cadastro de Item Onde o usuário pode cadastrar um novo item para doação, inserindo título, descrição, categoria e imagens.

Tela de Perfil Apresenta informações do usuário, histórico de itens doados e selos/conquistas obtidas por doações realizadas.

Jornada Principal do Usuário 1-O usuário abre o aplicativo e acessa a tela inicial (Home/Feed).

2-Ele navega pelos itens disponíveis ou utiliza a tela de filtros para refinar a busca.

3-Ao encontrar um item de interesse, acessa a tela de detalhes.

4-Se tiver interesse, inicia uma conversa na tela de chat com o doador.

5-No chat, combina informações como disponibilidade e local de retirada.

6-Após o acordo, o usuário conclui a retirada/doação do item.

7-A doação concluída pode gerar um selo no perfil do usuário, incentivando novas interações.
